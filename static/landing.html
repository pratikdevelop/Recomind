How it works

From document to recommendation
in seconds

01
📂
Upload your docs
Drop in PDFs, Word files, CSVs, Markdown — any format. RecoMind chunks and embeds them automatically.
02
🔍
Vector search
Your query is embedded and matched against all chunks using cosine similarity across 768 dimensions.
03
⚖️
Re-rank for precision
A CrossEncoder model re-scores the top 15 candidates and keeps only the 3 most relevant chunks.
04
⭐
Smart recommendation
A local LLM synthesises the best answer with confidence score, reasoning, and alternatives.
